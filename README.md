# Snet: A simple deep learning library

A simple deep learning library which contains basic functions for creating a linear model, loading data and training the model.

## Components of the library

1. Tensors
2. Loss Functions
3. Layers
4. Neural Nets
5. Optimizers
6. Data
7. Training
8. Examples
